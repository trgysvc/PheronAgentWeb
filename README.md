# Pheron Agent Web

Pheron Agent Web is the web interface for the Pheron Agent ecosystem. Built with modern web technologies, this project serves as the frontend application for agentic workflows and ecosystem management.

## 🚀 Getting Started

First, make sure you have installed the necessary dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the application.

*Note: The project is currently configured to use `--webpack` in the dev script to ensure stability during local development.*

## 🛠 Tech Stack

- **Framework:** [Next.js](https://nextjs.org) (App Router)
- **Language:** TypeScript
- **Styling:** CSS
- **Linting:** ESLint

## 📁 Project Structure

- `src/app/` - Next.js Application routes and pages
- `public/` - Static assets and configurations

## 💻 Available Scripts

- `npm run dev`: Starts the local development server.
- `npm run build`: Creates an optimized production build.
- `npm run start`: Starts the application in production mode.
- `npm run lint`: Runs ESLint to check for code quality and errors.
